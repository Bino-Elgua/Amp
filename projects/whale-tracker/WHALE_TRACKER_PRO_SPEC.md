### 2. ON-CHAIN SIGNAL GENERATION

**Signal Types:**
- **Accumulation Signals** 🟢
  - Whale buying large volumes
  - Holding for long periods
  - Confidence: 65-95%
  - Action: BUY signal for followers

- **Distribution Signals** 🔴
  - Whale dumping holdings
  - Moving to exchange
  - Confidence: 75-98%
  - Action: SELL signal / Take profits

- **Whale Clustering Signals** 🟠
  - Multiple whales buying same asset
  - Synchronized wallet movements
  - Confidence: 70-92%
  - Action: Strong directional signal

- **Smart Money Signals** 🟡
  - Track successful whale wallets (historical win rate 65%+)
  - Copy their moves in real-time
  - Confidence: 80-95%
  - Action: Mirror portfolio

**Data Points per Signal:**
- Transaction hash + timestamp
- Whale wallet address + historical win rate
- Asset (ETH, BTC, tokens)
- Amount + USD value
- Source/destination (wallet, exchange, DEX)
- Whale portfolio composition
- Historical behavior pattern
- Machine learning confidence score
- Predicted market impact
- Similar past patterns (with outcomes)
