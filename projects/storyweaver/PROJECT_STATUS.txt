API ENDPOINTS (23 TOTAL)
================================================================================

Health:
  GET /api/health                    Backend status
  GET /api/ollama/health             Ollama status

Chat:
  POST /api/chat                     Send message to AI

Stories:
  POST /api/stories                  Create new story
  GET  /api/stories/<id>             Get story details
  PUT  /api/stories/<id>             Update story

Books:
  POST /api/stories/<id>/generate-book        Generate EPUB
  POST /api/stories/<id>/send-to-kindle       Email to Kindle
  GET  /api/stories/<id>/download             Download book

Stripe (Payments):
  POST /api/stripe/checkout          Create checkout
  POST /api/stripe/webhook           Payment webhook

Full docs in DELIVERY.md

================================================================================
CUSTOMIZATION OPTIONS
================================================================================

Edit .env to:
  - Change AI model: OLLAMA_MODEL=mistral (any Ollama model)
  - Change port: WEBUI_PORT=8888
  - Enable monetization: MONETIZATION_ENABLED=true
  - Set Stripe keys: STRIPE_PUBLIC_KEY, STRIPE_SECRET_KEY
  - Enable images: GENERATE_ILLUSTRATIONS=true
  - Set Kindle email: KINDLE_EMAIL=you@kindle.com

================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

Problem: Ollama not starting
  → Check: pgrep ollama
  → Fix: ollama serve (manual start)

Problem: Port 8080 in use
  → Change WEBUI_PORT in .env
  → Restart: bash run.sh stop && bash run.sh start

Problem: Out of memory
  → Use lighter model: OLLAMA_MODEL=gemma2
  → Close other apps

Problem: Backend errors
  → Check logs: tail -f logs/backend.log
  → Check health: curl http://localhost:5000/api/health

For more: See SETUP_GUIDE.md Troubleshooting section
