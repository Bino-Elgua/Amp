## Overview
Fitness app with 3D avatar evolution. AI health coach, weight/workout tracking, gamification. Avatar morphs as user progresses. Detailed financial model projecting $395K Y1 revenue. 45 files.
## Tech Stack
- **Frontend:** Next.js 14, React, TypeScript
- **3D:** Three.js, React Three Fiber, Ready Player Me
- **Backend:** Supabase (PostgreSQL), NextAuth.js
- **Charts:** Recharts
- **Payments:** Stripe (planned)
