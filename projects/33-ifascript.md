## Overview
Esoteric programming language/VM inspired by Yoruba Ifá divination. 256 Odu mapped to 256 opcodes, cowrie casts as entropy oracle. Rust implementation, WASM-ready. 14 files. AGPL-3.0 licensed.
## Tech Stack
- **Core:** Rust (edition 2021)
- **Deps:** rand, reqwest, serde_json, hex
## Verdict
Beautiful cultural-tech project. Core IP for the Technosis ecosystem. No direct revenue (by design). ⭐⭐⭐ (3/5)
